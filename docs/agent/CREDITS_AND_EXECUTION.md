# Credits & Execution

## Regeln
- Landingpage: KEINE Credits
- Dashboard: Credits schätzen → Check → Abbuchen nach Erfolg
- Fehlgeschlagene Schritte: möglichst nicht berechnen
- Bei Fehler nach Abbuchung: zurückbuchen
- Retry: max 2-3x, Credits nur bei finalem Erfolg
